How to build a Hexo blog with tranquilpeak theme.

How to build a Hexo blog. 🏡

Jan 29, 2025 in tutorial

  • 안녕하세요, Blog 주인장 PETER입니다.👋
  • 이 블로그는 정적인(Static) Site를 만들어주는 Hexo라는 Node.js 기반 프레임워크로 만들어졌습니다. 한물간(?) 프레임워크이긴 한 것 같습니다만, 그래도 빠르게 이쁜 블로그를 뚝딱 만들 수 있다는 건 아무래도 큰 장점은 맞는 것 같아요. :) 만들다가 이왕 알게된 기능 설정 방법들에 대해 정리해 놓습니다.
  • 이 블로그에 적용된 Tranquilpeak 이라는 테마를 중점적으로 설명하려 하는데요, Hexo를 설치하고 github.io blog로 만드는 방법, Hexo 테마 설정하는 방법까지는 조금만 검색하셔도 금방 찾으실 수 있으니 아신다고 가정하고 작성할게요. :)
  • 나만의 홈페이지를 만든다는 것은 온라인에 나만의 공간, 나만의 집🏠을 만드는 것과도 같죠.😉 혹여나 이 블로그가 마음에 드셔서 비슷하게 만들어 보시고자 하는 분들께 도움이 되면 좋겠네요!
